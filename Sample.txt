This is a sample file created to learn git operations.

Somechanges are made to this file.

Editing the file after adding this file to the remote online repository.
After editing, we have to stage the file, commit it and then push the file to the online repository. 
The command sequences are: 

git add Sample.txt 
git commit -m "File Updated" 
git push -u origin main 

After every edit or completion of tasks, we will have to repeat the above commands to push the updated version to the online repository so that other users/colleagues can access the updated code file(s). 


File contents are changed by cloning it to another folder and making changes.
The changed contents will now be uploaded back to the online repository.
